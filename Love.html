How to Write Love in Chinese

With the help of HTML, CSS, and JavaScript

Introduction

Chinese characters can seem like gibberish to those unfamiliar with the language. Even a single word can seem like a mish-mash of random shapes and lines. However, with a little bit of assistance, it can be quite simple to break down a character line-by-line and make memorization a breeze. Take a look below to see how HTML, CSS, and Javascript can help us break down the Chinese word for love: 爱 (pronounced ài, like 'aye').

HTML

With just HTML, we cannot do much to help a reader new to Chinese learn how to write a character. The best we can do is simply display the character itself on the webpage.

Here it is in paragraph (<p>) form:

Making it a header (<h1>) makes it a bit easier to see:

This isn't very helpful. Although you could sit down and draw what you see over and over again, Chinese actually has a specific stroke order for each of its tens of thousands of characters. Just displaying the character doesn't show you the correct way it is written. We clearly need a bit of help.

CSS

Adding CSS allows us to play around with how our HTML text is styled. For instance, we can make the character much larger with help from the CSS font-size property. We can also make it a bit easier to see using the HTML <div> element as well as adding a border and background color via CSS:

爱

This is better, but we are still left with the same problem as before: stroke order. An image can help us begin to chip away at this issue. Here, we display the image using HTML and use CSS to add a border around it and some padding to make it stand out.

Success! Our stroke order problem has been dealt with. You can now follow the image to draw the character in the correct sequence. But wait, we still haven't addressed another quirk of Chinese characters. Not only does each character have a proper stroke order, but each stroke also must be written in a specific direction.

While we could add some arrows to the image above displaying the correct direction, there is a different method that will be far more useful to you. Let's break out a bit of JavaScript to make an animation.
